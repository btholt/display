Parallax

Package : com.danehansen.display
Class : public class Parallax
Inheritance : Parallax > Object

A class for parallaxing based on mouse position or device orientation. The root of the parallax object would be the parent housing all the children that should be parallaxed. This object could be document if desired. Along with the JavaScript instantiation, some markup is required as well to make the effect work. Any child nodes that are intended to be parallaxed should include either/both data-parallaxX and data-parallaxY attributes, which should be set as numbers representing the distance in pixels the child node will travel on that axis. Negative nubmers can be used to have a reversed parallax effect. Top and left css values are fine to have on the child nodes, as this effect uses translate3d to move the children. SVG nodes can have this applied as well.

Public Properties

  • speed : Number
    A number between 0 and 1 which represents the speed at which the parallax effect eases into its destination positioning. The default value is 0.2.

Public Methods

  • Parallax(element:Element, speed:Number = 0.2)
    Creates and activates a Parallax object, using the DOM element or jQuery object which the mouse positioning should be based on. An optional speed argument will immediately set the speed property.
  • pause():*
    Causes the Parallax instance to temporarity cease functioning.
  • resume():*
    Resumes a paused Parallax instance’s functionality.
  • destroy():*
    Cleans up the instance when it is no longer needed.

EXAMPLE 1: BASIC PARALLAX

var parallax=new Parallax(document.getElementById("parallax_wrapper"));
